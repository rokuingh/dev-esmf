\subsubsection{ESMF\_AllocFlag}
 
{\sf DESCRIPTION:\\}


Valid values are:
\begin{description}
   \item [ESMF\_DO\_ALLOCATE]
         The framework allocates the data space.
   \item [ESMF\_NO\_ALLOCATE]
         No data space allocation.
\end{description}
